# clj-test-website

A Clojure Website for learning and testing purposes.

## Usage

# Install Leiningen
http://leiningen.org/

# Run:
"lein cljsbuild auto"
"lein ring server"

# For REPL access:
After the website opened up, 
launch "lein trampoline cljsbuild repl-listen",
and make sure to refresh the page or commands will stall.
